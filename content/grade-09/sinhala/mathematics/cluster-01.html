සංඛ්‍යා රටා

රටාවක සාධාරණ පදය (Tₙ) සොයා, ඕනෑම පදයක් කල්තියා කීම.

මේ පාඩම අවසානයේ ඔබට පුළුවන් වෙයි 🎯
  • සංඛ්‍යා රටාවක සාධාරණ පදය (Tₙ) ලිවීමට
  • Tₙ ප්‍රකාශනය යොදා ඕනෑම පදයක් සෙවීමට
  • රටා ඇතුළත් ‘වචන ගැටළු’ විසඳීමට
1.1 සංඛ්‍යා රටාවක සාධාරණ පදය ›
1 හිතන්න…
අවුරුදු මේසෙට අම්මා කැවුම් අතුරනවා: පළවෙනි පේළියේ 5යි, ඊළඟට 11යි, ඊළඟට 17යි… මල්ලී අහනවා “100 වෙනි පේළියට කීයද?” පේළි 100ක් අතුරන්නෙ නැතුව — සූත්‍රයකින් හොයාගන්න පුළුවන්! ඒකයි අද ඉගෙන ගන්නේ.
2 සරල විදිහට

රටාවක එක එක සංඛ්‍යාවට කියන්නේ පදයක් (term). ඒවා නම් කරන්නේ මෙහෙමයි: පළමු පදය T₁, දෙවැනි T₂, තුන්වැනි T₃ … n වෙනි පදය Tₙ. මේ Tₙ ට කියන්නේ සාධාරණ පදය (general term). මෙහි n යනු පදයේ ස්ථානයයි — n කවදත් ධන නිඛිලයක් (1, 2, 3 …); ‘1.5 වෙනි පදයක්’ නෑ!

3 ඇසට පෙනෙන්න

හැම පදයක්ම කලින් එකට 6 බැගින් වැඩි වෙනවා. මේ ‘පොදු වෙනස’ට කියන්නේ d = 6.

4 සාධාරණ පදය හොයමු
රටාව: 5, 11, 17, 23, …
පොදු වෙනස d = 6 → Tₙ = 6n + ? T₁ = 5 නම්: 6(1) + ? = 5 → ? = −1 Tₙ = 6n − 1 පරීක්ෂාව: n=3 → 6(3)−1 = 17 ✓   n=4 → 23 ✓
ඉක්මන් සූත්‍රය: පොදු වෙනස d හොයලා → Tₙ = d·n + (T₁ − d). මෙතන 6n + (5−6) = 6n − 1!
✏️ ඔබේ වාරය: 3, 7, 11, 15, … රටාවේ Tₙ සොයන්න.
පිළිතුර බලන්න
d=4, T₁=3 → Tₙ = 4n + (3−4) = 4n − 1. පරීක්ෂාව: n=3 → 11 ✓
5 පරිස්සමින්!
⚠ d·n න් නවතින්න එපා
Tₙ = dn ලෙස නවතින්න එපා — (T₁ − d) කොටසත් එකතු කරන්න ඕනේ! 5,11,17 → 6n නෙවෙයි, 6n − 1.
6 ඉක්මන් පරීක්ෂාව
✅ ඉක්මන් පරීක්ෂාව
  1. 2, 5, 8, 11, … හි Tₙ?
    පිළිතුර
    d=3, T₁=2 → 3n − 1
  2. 10, 20, 30, … හි Tₙ?
    පිළිතුර
    d=10, T₁=10 → 10n + 0 = 10n
🔑
d හොයන්න → Tₙ = d·n + (T₁ − d).
1.2 n වෙනි පදය සඳහා ප්‍රකාශනයක් ›
2 අනික් පැත්ත

දැන් Tₙ දීලා තියෙනවා නම්, n වෙනුවට 1, 2, 3 … දාලා රටාව හදන්නත්, ඕනෑම පදයක් සොයන්නත් පුළුවන්.

4 වැඩ කළ උදාහරණය (පොත I, පි. 2)
Tₙ = 2n + 3 රටාවේ පළමු පද 4 හා 20 වෙනි පදය
n=1,2,3,4 → 5, 7, 9, 11 20 වෙනි පදය: T₂₀ = 2(20) + 3 = 43
✏️ ඔබේ වාරය: Tₙ = 3n + 2 රටාවේ 10 වෙනි පදය?
පිළිතුර බලන්න
T₁₀ = 3(10)+2 = 32
5 පරිස්සමින්!
⚠ ‘අගය’ ද ‘ස්ථානය’ ද?
“කවර පදය 50ට සමානද?” ඇහුවොත් — සමීකරණයක් (Tₙ = 50) හදලා n හොයන්න! “20 වෙනි පදයේ අගය?” ඇහුවොත් — n=20 දාලා Tₙ හොයන්න.
7 මතක උපක්‍රමය
🔑
n = ස්ථානය (කීවෙනිද). Tₙ = ඒ ස්ථානයේ අගය. පදය හොයන්න n දාන්න; ස්ථානය හොයන්න සමීකරණයක්.
1.3 රටා ඇතුළත් ගැටළු විසඳීම ›
1 හිතන්න…
මේසයකට පුටු 4ක්. මේස 2ක් එකට තිබ්බම පුටු 6ක්, 3ක් එකට 8ක්… මේස n කට කීයද? මේ ‘කතාව’ රටාවකට හරවලා සූත්‍රයකින් විසඳමු!
4 වැඩ කළ උදාහරණය
මේස 1→4, 2→6, 3→8 … පොදු වෙනස d=2 Tₙ = 2n + (4−2) = 2n + 2 මේස 10කට: T₁₀ = 2(10)+2 = 22 පුටු
✏️ ඔබේ වාරය: ටයිල් මෝස්තරයක රූප 1ට ටයිල් 5, රූප 2ට 9, රූප 3ට 13… රූප 8ට කීයද?
පිළිතුර බලන්න
d=4, T₁=5 → Tₙ = 4n+1. රූප 8: 4(8)+1 = 33
🧮 සාධාරණ පද යන්ත්‍රය — Tₙ = a·n + b
Tₙ =n + · n =
a, b සහ n දී "පදය සොයන්න" ඔබන්න — රටාවත් පෙන්වයි.

🔑
කතාව → රටාව → d හොයන්න → Tₙ සූත්‍රය → අගය දාන්න.

විභාගයට පෙර මේ පිටුව විතරක් බැලුවත් ඇති.

1 · සංඛ්‍යා රටා

Tₙ = සාධාරණ පදය, n = ස්ථානය.
d හොයන්න → Tₙ = d·n + (T₁ − d).
අගය → n දාන්න; ස්ථානය → සමීකරණයක්.

2 · ද්විමය

ස්ථාන අගය = 2 හි බල (1,2,4,8…).
දශම→ද්විමය: 2න් බෙදා ඉතිරි යටින් උඩට.
1+1=10; ණයට ගත්තම 2ක්.

3 · භාග

‘න්’ = ×.

÷,× — වමේ සිට දකුණට.
